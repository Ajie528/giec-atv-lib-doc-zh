## 模块介绍

### WiFi 介绍

在Android开发过程中，我们可能会使用到WiFi相关的操作，例如开启或关闭WiFi、断开或者连接WiFi、移除WiFi、获取WiFi IP、开启或者关闭WiFi 热点等操作。
本模块对上述功能都做了封装，方便开发者能快速开发应用，提高工作效率。

> 引入 `import com.giec.stb.core.net.wifi.*`



### WifiUtil 使用说明

> **重要提示**：由于Android系统的特性，当设备通过以太网连接时，WiFi无法主动建立连接。
>
> 但是WiFi网络配置会被保存为"已保存"状态，当以太网连接断开时，设备会自动连接到已保存的WiFi网络。

```kotlin
// 开启 WiFi
WifiUtil.openWifi(context)

// 关闭 WiFi
WifiUtil.closeWifi(context)

// 检查 WiFi 是否已启用
WifiUtil.isWifiEnabled(context)

// 断开当前连接的 WiFi
WifiUtil.disconnectCurrentWifi(context)

// 强制重新连接 WiFi
WifiUtil.forceReconnect(context)

// 连接开放 WiFi 网络
WifiUtil.connectWifi(context, ssid)

// 连接隐藏的开放 WiFi 网络
WifiUtil.connectWifi(context, ssid, isHidden = true)

// 连接有密码的 WiFi（自动检测安全类型 - 推荐）
WifiUtil.connectWifiAutoDetect(context, ssid, password)

// 连接有密码的 WiFi（指定安全类型）
WifiUtil.connectWifi(context, ssid, password, WifiSecurityType.WPA2_PSK)

// 连接隐藏的有密码 WiFi
WifiUtil.connectWifi(context, ssid, password, WifiSecurityType.WPA2_PSK, true)

// 移除已保存的 WiFi 配置
WifiUtil.removeWifi(context, ssid)

// 获取当前连接的 WiFi SSID
WifiUtil.getConnectedSsid(context)

// 获取 WiFi IP 地址
WifiUtil.getWifiIp(context)
```

### 安全类型支持

```kotlin
enum class WifiSecurityType {
    OPEN,       // 开放网络（无密码）
    WEP,        // WEP加密
    WPA_PSK,    // WPA个人版
    WPA2_PSK,   // WPA2个人版（最常用）
}
```

### 使用示例

```kotlin
// 基本使用：连接到开放WiFi
if (WifiUtil.connectWifi(context, "公共WiFi")) {
    val ip = WifiUtil.getWifiIp(context)
    println("连接成功！IP地址：$ip")
}

// 连接到密码保护的WiFi（推荐使用自动检测）
val success = WifiUtil.connectWifiAutoDetect(context, "家庭网络", "我的密码")
if (success) {
    println("连接成功！当前SSID：${WifiUtil.getConnectedSsid(context)}")
}

// 连接到隐藏网络
WifiUtil.connectWifi(context, "隐藏网络", "密码123", isHidden = true)

// 完整的连接流程
fun connectToWiFi(ssid: String, password: String?) {
    // 确保WiFi已开启
    if (!WifiUtil.isWifiEnabled(context)) {
        WifiUtil.openWifi(context)
    }
    
    // 连接网络
    val success = if (password != null) {
        WifiUtil.connectWifiAutoDetect(context, ssid, password)
    } else {
        WifiUtil.connectWifi(context, ssid)
    }
    
    if (success) {
        showToast("成功连接到 $ssid")
    } else {
        showToast("连接失败，请检查网络设置")
    }
}
```

### 注意事项

- **连接时间**：普通网络约3-5秒，隐藏网络约5-10秒
- **自动检测**：`connectWifiAutoDetect` 会按安全性从高到低尝试不同协议，更加可靠
- **生命周期**：工具类会自动处理Activity销毁等情况，避免应用崩溃
- **返回值**：所有连接方法都返回布尔值，表示连接是否成功

这个增强版的WiFi工具类提供了更加稳定和功能丰富的WiFi连接管理，适合各种复杂的网络环境。

```kotlin


// 连接有密码的的 WiFi
WifiUtil.connectWifi(context, ssid, password)
WifiUtil.connectWifi(context, ssid)

// 移除已保存的 WiFi
WifiUtil.removeWifi(context, ssid)

// 获取 WiFi IP
WifiUtil.getWifiIp(context)

```

### WifiHotspotUtil 使用

#### 参数说明
| 参数                   | 名称                    |
|----------------------|-----------------------|
| ssid                 | WiFi 热点的名称            |
| password             | WiFi 热点的密码            |
| band | 频段  1: 2.4GHz，2: 5GHz |
| autoShutdownWhenIdle | 是否在空闲时停止WiFi热点, 默认关闭  |
| hidden               | 是否隐藏SSID, 默认关闭        |


```kotlin

// 开启  2.4G WiFi 热点
WifiHotspotUtil.startTethering(context, ssid, password, band = 1, autoShutdownWhenIdle, hidden)

// 开启 5G WiFi 热点
WifiHotspotUtil.startTethering(context, ssid, password, band = 2, autoShutdownWhenIdle, hidden)

// 关闭 WiFi 热点
WifiHotspotUtil.stopTethering(context)

// 设置是否自动关闭WiFi热点
WifiHotspotUtil.setAutoShutdownEnabled(context, enable)

// 清除所有网络接口，保留eth0和wlan0
WifiHotspotUtil.clearAllNet()

// 将热点设置为Trunk模式
WifiHotspotUtil.setTrunkMode(vlanIface, bridgeIface)

// 将热点设置为桥接模式
WifiHotspotUtil.setBridgeMode()

// 将 STA 模式设置为静态 IP
WifiHotspotUtil.setStaModeToStaticIP(ipCidr, gateway, dns1, dns2)

// 将 STA 模式设置为动态 IP
WifiHotspotUtil.setStaModeForToDynamicIP()

// 获取当前热点SSID
getCurrentHotspotSSID(context)

// 获取热点密码
getCurrentHotspotPassword(context)

// 获取网络配置信息
getNetworkConfigInfo()
```
#### 例子：
```kotlin
// 1. 清除所有网络接口，保留eth0和wlan0
WifiHotspotUtil.clearAllNet()

// 2. 启动 WiFi 热点
WifiHotspotUtil.startTethering(context, "GIEC", "12345678")

// 3. 等待ap接口完成创建
Thread.sleep(3000)

// 4.设置为 Trunk 模式
WifiHotspotUtil.setTrunkMode("eth0.100", "br-vlan100")
```
